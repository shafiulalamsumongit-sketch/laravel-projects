
# Why type="module" is required
Modern Vue (Vue 3) uses ES Modules.
That means:
import ... from 'vue'
export default {}

➡️ Browsers only understand this when:
<script type="module"></script>
Laravel Vite automatically outputs ES modules, so type="module" is mandatory.

# Correct Laravel + Vue Setup (Recommended)

Install Vue with Vite
composer create-project laravel/laravel app
cd app
npm install
npm install vue @vitejs/plugin-vue
